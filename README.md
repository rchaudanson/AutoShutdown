
# AutoShutdown
Autoshutdown is a small application that allows a user's computer to be shut down at a predefined time. Easily from the interface, the user can cancel the scheduled shutdown of his computer at any time for the current day, the automatic shutdown is reactivated the next day. The goal is to turn off unused computers that have not been turned off by users.

Autoshutdown est une petite application qui permet d’éteindre l’ordinateur d’un utilisateur à une heure prédéfinie.
Facilement depuis l’interface, l’utilisateur peut annuler à tout moment l’arrêt programmé de son ordinateur pour la journée en cours, l’arrêt automatique est réactivé le lendemain.
Le but étant d’éteindre les ordinateurs non utilisés qui n’ont pas été éteints par les utilisateurs.
